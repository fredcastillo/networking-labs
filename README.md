## 📘 Sobre este repositorio

Este repositorio contiene mis **laboratorios de redes**, desarrollados como parte de mis estudios y práctica técnica mientras curso un **Tecnólogo en Seguridad Informática**.

El objetivo de estos proyectos es diseñar, configurar y analizar **infraestructuras de red de estilo empresarial**, documentando configuraciones reales y el comportamiento de diferentes protocolos de red.

Cada laboratorio busca demostrar la implementación práctica de conceptos como **enrutamiento, servicios de red, segmentación y resolución de problemas (troubleshooting)**.

Estos proyectos forman parte de mi **portafolio técnico enfocado en redes y ciberseguridad**.

---

## 🛠 Tecnologías y herramientas

Dependiendo del laboratorio, se utilizan diferentes herramientas y entornos de simulación, como:

- Plataformas de simulación o emulación de redes
- Protocolos de enrutamiento dinámico (OSPF y otros en el futuro)
- Segmentación de red con VLAN
- Servicios DHCP
- Conectividad WAN
- Comandos de verificación y troubleshooting

Los laboratorios pueden implementarse usando plataformas como **Cisco Packet Tracer, GNS3 u otros entornos de simulación de redes**.

## 📂 Proyectos del Repositorio

### 🌐 Hub-Spoke OSPF DHCP Network

Laboratorio que simula una **red empresarial multi-sitio** utilizando una topología **Hub-and-Spoke**.

Incluye la implementación de:

- Enrutamiento dinámico con OSPF
- Servicio DHCP centralizado
- Conectividad WAN mediante enlaces seriales
- DHCP relay utilizando `ip helper-address`

📁 Carpeta del proyecto: `hub-spoke-ospf-dhcp-network`  
📄 La documentación completa se encuentra en el README del proyecto.
