## 📘 About this repository

This repository contains my **networking labs**, developed as part of my studies and hands-on practice while pursuing a **Technologist Degree in Information Security**.

The goal of these projects is to design, configure, and analyze **enterprise-style network infrastructures**, documenting real configurations and the behavior of different networking protocols.

Each lab aims to demonstrate the practical implementation of concepts such as **routing, network services, segmentation, and troubleshooting**.

These projects are part of my **technical portfolio focused on networking and cybersecurity**.

---

## 🛠 Technologies and tools

Depending on the lab, different tools and simulation environments are used, such as:

- Network simulation or emulation platforms
- Dynamic routing protocols (OSPF and others in the future)
- Network segmentation with VLANs
- DHCP services
- WAN connectivity
- Verification and troubleshooting commands

Labs may be implemented using platforms such as **Cisco Packet Tracer, GNS3, or other network simulation environments**.

## 📂 Repository Projects

### 🌐 Hub-Spoke OSPF DHCP Network

Lab that simulates an **enterprise multi-site network** using a **Hub-and-Spoke topology**.

It includes the implementation of:

- Dynamic routing with OSPF
- Centralized DHCP service
- WAN connectivity using serial links
- DHCP relay using `ip helper-address`

📁 Project folder: `hub-spoke-ospf-dhcp-network`  
📄 Full documentation is available in the project's README.
